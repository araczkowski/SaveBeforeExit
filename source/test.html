By default this plug-in is never reverting any change even if you decided to cancel the navigation.

To enable the reverting you only need to add a jQuery selector for items to rever
The Ignore Modifications Selector setting accepts a jQuery selector used to include elements to the restore list after cancel the navigation.
The following jQuery selector could be used to exclude a rich text editor:

#P25_VACATION_0,#P25_VACATION_1
